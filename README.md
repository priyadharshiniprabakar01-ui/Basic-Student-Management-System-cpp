Student Management System (C++)

Overview

A simple console-based Student Management System developed in C++. This project allows users to manage student records through a menu-driven interface.

Features

- Add new student records
- Display all student records
- Search student records using Roll Number
- Update existing student information
- Delete a specific student record
- Delete all records
- User-friendly menu-driven interface

Technologies Used

- C++
- Standard Template Library (STL)
- Console Application

Project Structure

The project stores:

- Student Name
- Roll Number
- Course
- Class/Section
- Contact Number

How to Run

1. Clone the repository:
   git clone <repository-link>

2. Open the project in any C++ IDE or compiler.

3. Compile and run:
   g++ main.cpp -o student_management
   ./student_management
